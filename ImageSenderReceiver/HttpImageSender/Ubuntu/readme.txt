Программа для отправки JPEG изображений по HTTP соединению.
Программа проверена на Ubuntu LTS 20.04.06.

Зависимости:
Для запуска может понадобиться набор библиотек Qt5. В ubuntu и debian можно установить командой
sudo apt install qt5-default


Запуск:

1. Открыть терминал
2. Перейти в дерикторию с программой
3. (Только при первом запуске)
	3.1 Выполнить sudo chmod +x HttpImageSender
	3.2 Ввести свой пароль
4. Выполнить ./HttpImageSender

Работа с программой:
0. Поднять сервер, который будет принимать изображение (например ServerExample, поставляемый вместе с программой)
1. В строке "URL" ввести адрес сервера (например http://127.0.0.1:8000)
2. В строке "Изображение" нажать кнопку "Выбрать" и выбрать JPEG файл
3. В этой же строке нажать кнопку "Отправить"

Выбранный файл отправится на сервер с указанным адресом.

В программе присутствует счетчик изображений, который увеличивается после каждой отправки. Он является номером отправляемого изображения (number).


Формат запроса:
Отправка осуществляется по HTTP (не HTTPS) соединению. Формат запроса:

POST / HTTP/1.1
Host: 127.0.0.1:8000
Content-Type: multipart/form-data; boundary=--------------------------052781154892520883617815
Mime-Version: 1.0
Content-Length: 114710
Connection: Keep-Alive
Accept-Encoding: gzip, deflate

--------------------------052781154892520883617815
Content-Disposition: form-data; name="number"
Content-Type: text/plain

--------------------------052781154892520883617815
Content-Disposition: form-data; name="image"; filename="image.jpeg"
Content-Type: image/jpeg

--------------------------052781154892520883617815--
