Программа для получения JPEG изображений по HTTP соединению.
Программа проверена на Windows 10 и Ubuntu LTS 20.04.06.

Зависимости:
Для запуска потребуется:
- Python3 (протестировано на 3.8.10)
- Модуль http.server для Python3

Запуск:

1. Открыть терминал Linux или командную строку Windows
2. Перейти в директорию с программой
3. Выполнить команду python3 main.py

Работа с программой:
1. Запустить программу. Она поднимет HTTP сервер по адресу 127.0.0.1:8000
2. С помощью стороннего клиента (например, HttpImageSender) отправить JPEG изображение в бинарном формате
3. Полученное изображение сохранится в директорию ./images/ 
Дополнительно в терминал будет выведено значение номера переданного изображения (number)


Формат запроса клиента:
Отправка осуществляется по HTTP (не HTTPS) соединению. Формат запроса:

POST / HTTP/1.1
Host: 127.0.0.1:8000
Content-Type: multipart/form-data; boundary=--------------------------052781154892520883617815
Mime-Version: 1.0
Content-Length: 114710
Connection: Keep-Alive
Accept-Encoding: gzip, deflate

--------------------------052781154892520883617815
Content-Disposition: form-data; name="number"
Content-Type: text/plain

--------------------------052781154892520883617815
Content-Disposition: form-data; name="image"; filename="image.jpeg"
Content-Type: image/jpeg

--------------------------052781154892520883617815--
